# Privacy Statement

LabLens processes laboratory documents entirely within the browser whenever possible. Files and parsed data are stored locally using IndexedDB/sql.js. Cloud OCR and LLM normalization are opt-in features that require explicit consent. No personal data leaves the device unless you enable a cloud option.

## Data Handling
- **Uploads**: Processed in web workers and discarded after extraction.
- **Exports**: Generated PDFs exclude metadata and embed the safety disclaimer.
- **Telemetry**: Disabled by default; enabling sends only anonymous performance metrics.

## Rights
LabLens is designed for GDPR and Swiss FADP compliance. Users can clear data via browser storage controls. No third-party trackers are included.
