# Security Policy

## Supported Versions
LabLens is pre-release software. Security fixes are applied on `main`.

## Reporting a Vulnerability
Email security reports to security@example.com with the subject `LabLens`. Expect an acknowledgement within 72 hours. Please include reproduction steps and impact assessment.

## Hardening Checklist
- [x] Strict CSP headers
- [x] PHI scrubbing prior to rendering
- [x] No third-party analytics
- [x] Optional cloud features disabled by default
